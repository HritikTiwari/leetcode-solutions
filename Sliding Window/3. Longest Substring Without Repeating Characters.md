# 3. Longest Substring Without Repeating Characters

## Pattern

Sliding Window

## Problem

Given a string `s`, return the length of the longest substring without repeating characters.

A substring is a contiguous sequence of characters.

---

## Constraints

```text
0 <= s.length <= 5 × 10^4
s consists of English letters, digits, symbols and spaces.
```

---

# Approach 1: Brute Force

## Idea

Start from every character and keep extending the substring until a duplicate character is found.

Use a HashSet to keep track of the characters already present in the current substring.

---

## Algorithm

1. Pick every possible starting index.
2. Create an empty HashSet.
3. Expand the substring one character at a time.
4. If a duplicate character is found, stop expanding.
5. Update the maximum length.

---

## Code

```python
class Solution:
    def lengthOfLongestSubstring(self, s: str) -> int:
        maximum = 0
        n = len(s)

        for i in range(n):
            my_set = set()

            for j in range(i, n):

                if s[j] in my_set:
                    break

                maximum = max(maximum, j - i + 1)
                my_set.add(s[j])

        return maximum
```

## Complexity Analysis

### Time Complexity

```text
Outer Loop : O(n)

Inner Loop : O(n)

Overall : O(n²)
```

### Space Complexity

```text
O(n)
```

because the HashSet may store all unique characters.

# Why Improve?

Notice that when we move to the next starting position, we repeat many of the same comparisons.

Example:

```text
abcabc
```

We repeatedly examine almost the same substring.

Can we reuse the previous work instead of starting over?

Yes.

---

# Approach 2: Sliding Window (Optimal)

## Key Observation

Instead of restarting from every index, maintain a window containing only unique characters.

If a duplicate character is found:

* Move the left pointer just after the previous occurrence of that character.
* Continue expanding the right pointer.

This allows us to reuse the previous window.

---

## Why Sliding Window?

Think of the window as:

```text
abcabcbb

^
left

      ^
     right
```

The window always contains unique characters.

Whenever a duplicate appears:

```text
abcabc

   ^
Duplicate 'a'
```

Move the left pointer:

```text
abcabc

  ^
left

```

until the duplicate is removed.

Instead of rebuilding the substring, we simply adjust the window.

---

## Algorithm

1. Initialize:

   * `left = 0`
   * `right = 0`
   * `maximum = 0`
2. Store the last index of every character in a HashMap.
3. Expand the window using `right`.
4. If the current character already exists inside the current window:

   * Move `left` to one position after its previous occurrence.
5. Update the maximum window size.
6. Store the latest index of the character.
7. Continue until the end of the string.

---

## Code

```python
class Solution:
    def lengthOfLongestSubstring(self, s: str) -> int:
        left = 0
        right = 0
        maximum = 0

        my_dict = {}

        while right < len(s):

            if s[right] in my_dict:
                left = max(left, my_dict[s[right]] + 1)

            maximum = max(maximum, right - left + 1)

            my_dict[s[right]] = right

            right += 1

        return maximum
```


## Why `left = max(left, lastIndex + 1)`?

Suppose:

```text
abba
```

Current window:

```text
ab

left = 0
```

Next character:

```text
b
```

Previous index:

```text
1
```

Move left:

```text
left = 2
```

Now the window becomes:

```text
ba
```

Notice we use:

```python
left = max(left, lastIndex + 1)
```

because the previous occurrence of the character may already lie outside the current window.

This prevents moving the left pointer backward.

---

## Complexity Analysis

### Time Complexity

```text
Each character enters the window once.

Each character leaves the window at most once.

Overall : O(n)
```

### Space Complexity

```text
O(n)
```

for storing the last index of each character.

# Learnings

* Sliding Window is useful for contiguous subarray or substring problems.
* A HashMap helps remember the last occurrence of each character.
* The left pointer never moves backward.
* `left = max(left, lastIndex + 1)` prevents revisiting characters outside the current window.
